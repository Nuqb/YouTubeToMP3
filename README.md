## ğŸš€ Quick Start

### Prerequisites
- Python 3.7+
- FFmpeg

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/Nuqb/youtubeToMP3.git
   cd youtubeToMP3
   ```

2. **Install FFmpeg**
   ```bash
   # Ubuntu/Debian
   sudo apt install ffmpeg
   
   # macOS
   brew install ffmpeg
   
   # Windows - Download from https://ffmpeg.org
   ```

3. **Install dependencies**
   ```bash
   pip install -r requirements.txt
   ```

4. **Run the app**
   ```bash
   python app.py
   ```

5. **Open your browser**
   ```
   http://localhost:5000
   ```
